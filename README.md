## 📌 Project Overview
Designing and implementing a cost-effective Security Operations Center (SOC) 
for MedCore Pharmacy Ltd — a fictional 100–250 user pharmaceutical SMB. 
Built using Microsoft Sentinel, Hybrid Active Directory, and aligned to 
HIPAA Security Rule compliance requirements.

---

## 🎯 Objectives
- Deploy a production-grade hybrid AD + Azure AD identity infrastructure
- Implement Microsoft Sentinel as the core SIEM/SOAR platform
- Map all detections to MITRE ATT&CK framework
- Automate alert triage, ticketing, and response via Logic Apps
- Integrate AI triage using Microsoft Copilot for Security
- Build proactive threat intelligence pipeline (MISP + MDTI)
- Achieve HIPAA + NIST CSF + CIS Controls compliance mapping
- Deliver full cost-benefit analysis for SMB SOC deployment
